Lists allow for you to be able to show information in a highly legible format.

The ordered list allows for you to show items in a numbered pattern:
  1. First item in a list
  2. Second item in a list
  3. Third item in a list
  4. Fourth item in a list
The unordered list allows for you to show items with a simple bullet point next to them:
  • First item in a list
  • Second item in a list
  • Third item in a list
  • Fourth item in a list
You can even nest lists!

    • This
